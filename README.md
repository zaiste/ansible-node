ansible-node
=========

Ansible role that installs Node for Ubuntu Xenial from the NodeSource APT.

Requirements
------------

None.

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: zaiste.node }
